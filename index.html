Ms. Pac-Man: A Genetic Programming Approach

Ms. Pac-Man searches for pills while trying to avoid the ghosts; the ghosts however want to track her down and eat her!


Random Search:
both Ms. Pac-Man and the ghosts use random search
Genetic Programming Search:
Ms. Pac-Man uses genetic programming. The ghosts use random search.
Competitive Coevolutionary Search:
Both Ms. Pac-Man and the ghosts use genetic programming trained in a competitive coevolution.
